# 0425-Homework
0425 Homework

此作業需正確設定電子郵件帳號及應用程式密碼，否則將無法正常運作。
請依下列步驟操作：<br>
1. 開啟專案根目錄下的 config.php 檔案。<br>
2. 將 mail_username 改為您的電子郵件帳號（例如：example@gmail.com）。<br>
3. 將 mail_password 改為您的電子郵件服務提供的應用程式專用密碼。
